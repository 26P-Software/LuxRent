Module 2: CSS Grid Layout Engine

Core Use Case: Complex, Two-Dimensional Page Layouts (Rows AND Columns Together)

1. Structural Mental Model

Unlike Flexbox, which deals with individual linear paths, Grid allows you to predefine columns and rows beforehand, mapping child elements onto the resulting matrix.

  • Grid Tracks: Generic terms indicating an entire singular path row or column row.
  • Grid Lines: Numbered structural boundaries dividing up coordinates. Crucially, numbering begins at index 1, not 0.
  • Grid Cell: A single intersection slot inside the system grid layout space.

2. Creating Columns and Rows

  • grid-template-columns: 200px 1fr 1fr;
    Defines 3 explicit columns. Column 1 is locked at 200px, while columns 2 and 3 split remaining fractions.
  • fr (Fractional Unit): Represents a dynamic fraction of available free storage space.
  • repeat(3, 1fr);
    Syntactical shorthand that expands to exactly 1fr 1fr 1fr.
  • gap: 1rem;
    Applies clear gutters between items uniformly without breaking external block borders.

3. Explicit Item Placement

Child elements can span multiple tracks or place themselves directly into specific line slots:

  • grid-column: span 2;
    Instructs an element to stretch across two visual column units seamlessly.
  • grid-row: 1 / 3;
    Instructs an element to stretch vertically from line 1 through line 3.
Teaching Tip: Emphasize to students that Grid Lines count the *edges* of cells. For a 3-column setup, there are 4 grid lines. This is why grid-column: 1 / 4; spans the entire length of a 3-column track.
