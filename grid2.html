Look closely at the CSS styling rules on the parent container. The layout matrix layout is explicitly self-documenting:

grid-template-areas: 
    "head  head  head"
    "side  main  rail"
    "foot  foot  foot";

Because the item blocks are assigned via grid-area: [name];, they snap perfectly into place based entirely on where their name is positioned inside the grid layout string blocks.
